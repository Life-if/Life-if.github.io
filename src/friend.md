## 🧑🏻‍💻 关于我
我是 `SeanVoyager` ，一名专注于智能算法落地的AI工程师，现就职于某大型泛互联网公司。我致力于将算法理论与工程实践相结合，解决真实场景业务痛点。平时会偶尔参与些开源工作或者做一些科研（water paper）保持技术敏感度，也是`Type-Moon`的长期读者。

## 🔭 研究方向与兴趣
我的研究兴趣涉及算法设计、深度学习应用与系统工程，特别关注深度学习技术在复杂工业场景中的创新与应用。研究兴趣包括但不限于以下方向：

- [x] **表示学习与自监督学习**：致力于探索高效的特征学习机制，为工业损伤检测等任务提供基础特征支持。

- [x] **图像与视频理解**：研究自动化巡检与安防监控等复杂场景下的视觉内容感知与信息理解，核心任务设计目标识别、事件检测与场景理解。

- [x] **视觉-语言大模型应用**：探索多模态大模型在实际场景中的适配与落地，研究涉及模型后训练、检索增强生成（`RAG`）以及多智能体协同等。

- [x] **模型推理优化**：专注于面向资源受限环境的模型压缩、加速与部署技术，实现轻量模型及紧凑大模型的高吞吐、低延迟推理。
